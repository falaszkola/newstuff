Modele

Co to są modele w informatyce?

Definicja modelu: model to uproszczony obraz rzeczywistości, który pozwala łatwiej zrozumieć daną rzecz, kosztem mniej ważnych informacji.

Case study:

Problem z danymi

Wyobraź sobie małpę, która chce zjeść banany.
Świat wokół niej jest skomplikowany, są tam np.: drzewa, inne małpy, deszcz.
Małpa nie zrozumie wszystkiego na raz więc robi Model.

Zasada:

Każdy model upraszcza rzeczywistość i wprowadza założenia

Model małpy

"Banany są na drzewie." Idź do drzewa = dostaniesz banany.
To jest model. Model to prosty pomysł na to, jak działa świat.

Lecz model nie pokazuje wszystkiego

Na drzewie może być tygrys lub ktoś już zjadł banany.

Model mówi:
drzewo = banany

Lecz rzeczywistość mówi czasem tak, czasem nie.

Wniosek

Model to prosty sposób myślenia o skomplikowanym świecie.
Pomaga podejmować decyzje, ale nie jest idealny i nie uwzględnia wszystkiego.
Każdy model kłamie, ale bywa użyteczny.
